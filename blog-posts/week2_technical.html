Padding and Borders and Margins...Oh My!
7/5/14

In this post, I'm going to discuss padding, borders, and margins. What are they? How do they work? How do you use them in your website design? To illustrate, let's take a look at the box model.

The Box Model
In an HTML document, all elements are represented as boxes. In CSS, the "box model" is used in design and layout. What is this box model anyway? It's basically a box that surrounds HTML elements. It's made up of four parts:
1. Content: Positioned in the middle of the box. This is where text and images appear. Can set a background color to it.
2. Padding: Surrounds the content. Background color of the box is applied to it. Can set size for top, right, left, and bottom.
3. Border: Surrounds the padding. Can specifically set a border color, width, and style (ex. dotted, dashed, solid).
4. Margin: Surrounds the border. It does not have a background color; it's totally transparent. Can set size for top, right, bottom, and left.

It is important to factor in all of these parts when calculating the full width of an element in CSS.
For example:

The total width of this element is 250px. Did you think it was 225px?
Let's break it down:

200px (width of content)
+ 10px (5px for left padding + 5px for right padding)
+ 10px (5px for left border + 5px for right border)
+ 30px (15px for left margin + 15px for right margin)
=250px

So,
Total element width = width + left padding + right padding + left border + right border + left margin + right margin

Hopefully this helps your understanding of padding, borders, and margins. Now get to work on that website you're building!
